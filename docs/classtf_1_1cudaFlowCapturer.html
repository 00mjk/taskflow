Detailed Description

class object to construct a CUDA graph through stream capture

A cudaFlowCapturer inherits all the base methods from tf::cudaFlowCapturerBase to construct a CUDA graph through stream capturer. This class also defines a factory interface tf::cudaFlowCapturer::make_capturer for users to create custom capturers and manages their lifetimes.
